What is this App?

This R-powered application combines traditional finance with machine learning to help investors allocate capital intelligently. Users input asset tickers (stocks, bonds, commodities), set the risk-free rate and horizon, then choose from three optimization models to generate optimal portfolios with visuals and statistics.
